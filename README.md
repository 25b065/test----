# 오늘의 감정 나무

간단한 정적 웹페이지로, 나뭇잎을 클릭해 오늘의 감정을 기록하고, 감사한 일을 최대 10개 적을 수 있습니다.

- 파일: `index.html`, `styles.css`, `script.js`
- 저장: 브라우저의 `localStorage`에 저장됩니다.
- 내보내기/가져오기: JSON으로 저장/불러오기 가능합니다.

사용 방법
1. `index.html`을 브라우저에서 엽니다.
2. 나뭇잎을 클릭하고 감정과 이유를 입력한 다음 저장하세요.
3. 원하는 색으로 잎 색을 직접 선택하거나 "감정으로 색 자동 지정" 버튼으로 감정에 맞는 기본 색을 적용할 수 있습니다.
3. 감사 항목을 최대 10개 입력하고 "감사 저장"을 누르세요.
4. 조언(위로/해결책/피드백)이 우측에 자동 생성됩니다.

개선 아이디어
- 서버 저장(로그인) 추가
- 감정별 상세 통계 및 히스토리
- 감정에 따른 애니메이션, 사운드 추가
